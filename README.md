# Smart Schedular

## Setup Instructions

### Backend (C# .NET)

1. Navigate to the backend folder:
   ```bash
   git clone https://github.com/theman-ojjas/Appsian-Assignment-1.git
   cd Backend
   
2. Install dependencies (if any are managed via npm for tooling):

   ```bash
   npm install

3. Build the project:

   ```bash
   dotnet build

4. Run the backend:

   ```bash
   dotnet run

### Frontend (React)
Navigate to the frontend folder:

```bash
cd frontend
npm install
npm start
